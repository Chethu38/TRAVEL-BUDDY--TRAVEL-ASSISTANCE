About Travel Buddy

Travel Buddy is a mini-project web app that helps you discover cities like Ballari, Hospet and Hampi. It shows curated places, food spots, live weather information, and Google Maps directions — all in one interface.

Tech stack: Flask, MongoDB, HTML, Tailwind CSS, and APIs (like OpenWeather). You can extend it later with user reviews, favourites, trip planning and more cities.
